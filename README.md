# hf_emotion_classifier

Assignment 2 for Advanced NLP. Involves fine-tuning and quantization for multi-class classification.


File Directories:
./training_scripts directory: Includes script for finetuning, PTQ, QAT and QLoRA.
hf_data.py and metrics.py: Contains helper functions to load and summarize metrics respectively.
22161_anlp_inference.ipynb: Contains the inference codes for the models.
